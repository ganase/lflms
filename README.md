# lflms
Management application of your little free library

## テスト手順
Mac
1.Open terminal
2. 
git clone <このリポジトリのURL>
cd lflms

3.
python3 -m venv .venv
source .venv/bin/activate

4.
python -m pip install --upgrade pip
python -m pip install -r requirements.txt

5.
python app.py
6.
http://127.0.0.1:5001
